# Consent-Based Keyboard Input Logger 

This project is a **Python Tkinter GUI application** that logs keyboard input entered **only within the application**, along with timestamps. It is designed strictly for **educational and ethical purposes**, demonstrating how keyboard events can be captured with user consent.

##  File Information
- **Language:** Python  
- **GUI Framework:** Tkinter  
- **Log File:** `keystroke_log.txt`
